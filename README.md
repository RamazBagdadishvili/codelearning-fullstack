# CodeLearning - рЃЦрЃљрЃарЃЌрЃБрЃџрЃў Front-End рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ ­ЪЄг­ЪЄф

> рЃўрЃАрЃгрЃљрЃЋрЃџрЃћрЃЌ рЃЋрЃћрЃЉ-рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљ рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ, рЃАрЃарЃБрЃџрЃўрЃљрЃЊ рЃБрЃцрЃљрЃАрЃЮрЃЊ!

## ­ЪЊІ рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ

**CodeLearning** рЃљрЃарЃўрЃА рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ Front-End рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА 500+ рЃџрЃћрЃЦрЃфрЃўрЃљрЃА HTML-рЃЊрЃљрЃю React-рЃЏрЃЊрЃћ, рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџ рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃА, рЃњрЃћрЃўрЃЏрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА, рЃЊрЃљ рЃџрЃўрЃЊрЃћрЃарЃЉрЃЮрЃарЃЊрЃА.

## ­Ъџђ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

### Backend
- **Node.js** + **Express.js** Рђћ API рЃАрЃћрЃарЃЋрЃћрЃарЃў
- **PostgreSQL** Рђћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ
- **JWT** + **bcrypt** Рђћ рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ
- **Helmet** + **CORS** + **Rate Limiting** Рђћ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ

### Frontend
- **React 18** + **TypeScript** Рђћ UI рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЎрЃў
- **Vite** Рђћ Build tool
- **Tailwind CSS** Рђћ рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ
- **Zustand** Рђћ State management
- **CodeMirror 6** Рђћ рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў
- **React Router 6** Рђћ рЃарЃЮрЃБрЃбрЃўрЃюрЃњрЃў

## ­Ъј« рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

- **рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃџрЃћрЃЦрЃфрЃўрЃћрЃЉрЃў** Рђћ рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃўрЃЌ рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃўрЃЌ
